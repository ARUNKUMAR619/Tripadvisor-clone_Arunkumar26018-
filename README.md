# Tripadvisor Clone (Frontend)

This is a frontend clone of the Tripadvisor website built using HTML and CSS.  
The project focuses on layout design, card structures, and clean UI styling.

---

## 🚀 Features

- Navigation bar with search layout
- Hero and promotional sections
- Recommendation cards (plans & experiences)
- Rating system using custom circle icons
- Interactive footer with dropdown selections
- Clean and structured layout design

---

## 🛠️ Built With

- HTML5  
- CSS3  
- Font Awesome (for icons)

---

## 🎯 Purpose

This project was created to practice:
- Real-world UI cloning  
- Layout structuring  
- CSS positioning and styling  
- Building visually clean interfaces
